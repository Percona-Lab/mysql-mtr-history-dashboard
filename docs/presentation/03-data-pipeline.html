Data Pipeline

From raw XML to
queryable metrics

Each build produces 19 JUnit XMLs with overlapping data. The pipeline parses, deduplicates, and transforms them into a clean Prometheus-native format.

19 JUnit XMLs
~5.1 MB per build
→
parse + merge
resolve 2,203 dups
→
build.json
~1.6 MB canonical
→
OpenMetrics
5 metric types
→
Prometheus
~27K series/build
